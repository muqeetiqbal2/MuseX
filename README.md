## Getting Started

### Clone Repository
```
git clone https://github.com/muqeetiqbal2/MuseX.git  
cd MuseX
```

### Install Dependencies
```
npm install  
OR pnpm install  
OR yarn install
```

### Start Development Server
```
npm run dev
```

---

## Production Build

### Build  
```
npm run build
```

### Preview 
```
npm run preview
```
