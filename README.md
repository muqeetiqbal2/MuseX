## Getting Started

### Clone Project
```bash
git clone https://github.com/muqeetiqbal2/MuseX.git
cd MuseX
```

### Install Dependencies
```bash
npm install
# or
pnpm install
# or
yarn install
```

### Start Dev Server
```bash
npm run dev
```

---

## Production Build

### Build
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## Project Structure

```
MuseX/
├─ public/                  # Static assets (images, icons, audio)
├─ src/                     # Project source
│  ├─ assets/               # Images, SVGs, media
│  ├─ components/           # UI components
│  ├─ styles/               # Global styles
│  └─ main.ts               # App entry point
├─ index.html               # Main HTML file
├─ package.json
├─ tsconfig.json
├─ tailwind.config.ts
└─ vite.config.ts
```
